# My neovim configuration

Plugin manager: Lazy

## Plugins

- Colorschemes: tokyonight, habamax
- Highlighting: treesitter
- LSP: mason, nvim-lspconfig, mason-lspconfig
- Terminal: toggleterm
- Keymap: which-key
- Completion: blink.cmp, autopairs
- Fileexplorer: neotree
- Dashboard: snacks
- Styling: lualine, bufferline
- Formatting: conform
